# 📈 Stock Market Trend Signal Prediction Using Machine Learning

## 📌 Project Overview

This project is a Machine Learning-based web application that predicts whether the stock market trend is bullish or bearish using historical stock market indicators and technical analysis features.

The application was developed as part of a Machine Learning Internship Assessment Project.

---

# 🚀 Features

- Real-time stock trend prediction
- Machine Learning-based classification
- Feature engineering using technical indicators
- Interactive Streamlit dashboard
- Feature importance visualization
- RSI market analysis
- Professional UI design

---

# 🧠 Machine Learning Models Used

- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier

Final Selected Model:
- Random Forest Classifier

---

# 📊 Technical Indicators Used

- RSI
- MACD
- EMA 10
- EMA 30
- Daily Return
- Volatility
- Open Price
- High Price
- Low Price
- Close Price
- Volume
